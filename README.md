Introduction
This project explores the mechanics, implementation, and performance evaluation of advanced string-matching algorithms, with a primary focus on the Boyer-Moore algorithm.

## Assignment Description
String matching is a fundamental problem in computer science that involves locating one or more occurrences of a pattern string within a larger text string. Rather than relying on brute-force comparisons that yield a worst-case time complexity of O(nm) , advanced algorithms lower this cost by using structural aspects of the pattern in a preprocessing phase to safely skip significant chunks of the text in the actual search phase 

This assignment is structured into three main analytical sections:
* **Part A (Boyer-Moore Algorithm):** Technical implementation of the Boyer-Moore algorithm utilizing both the Bad Character and Good Suffix heuristics, including an in-depth execution trace, output analysis, and an evaluation of its operational strengths and weaknesses
* **Part B (Variations of the Boyer-Moore Algorithm):** A study of at least three variations of the algorithm—such as the Boyer-Moore-Horspool, Turbo Boyer-Moore, and Sunday/Quick Search algorithms—exploring their core mechanics and comparing their computational complexities
* **Part C (Comparison with the KMP Algorithm):** An analytical breakdown of the Knuth-Morris-Pratt (KMP) algorithm, contrasting it against Boyer-Moore in terms of time complexity, space complexity, and specific practical application contexts.

## Objectives
The core objectives of this assignment are to:
* **Manipulate Data Structures and Algorithms:** Construct a fully correct, well-structured, and clearly commented implementation to solve string-matching problems
* **Perform Complexity Analysis:** Evaluate and compare the theoretical and empirical time and space complexities of different string-searching paradigms
